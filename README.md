## 三、插件主要功能：
* 文章页面和自定义单页面链接加入安全跳转中台
* 主题提供的模版页面链接可支持安全跳转中台跳转（需要自行开启）
* 自定义白名单，支持泛域名（V1.5.0版本之后）
* 兼容链接管理插件的链接数据，可以排除不需要加入的组别
* 自定义安全中台页面内容，多种模版可供选择
* 兼容编辑器超链接卡片插件的链接
* 支持白名单链接缓存刷新
* 自定义链接跳转提示信息
* 下载链接跳转时支持验证功能
